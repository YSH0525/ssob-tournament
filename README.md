# 🏸 S.S.O.B 대진표 관리 (Badminton Tournament Manager)

배드민턴 모임 대회 운영을 위한 웹 기반 대진표 관리 프로그램. 단일 HTML 파일로 서버 없이 브라우저에서 바로 동작.

## 주요 기능

### 👥 회원 관리
- 회원/비회원 구분 (콕 2/3개 규칙 자동 반영)
- 성별·연락처·가입일·메모·활성 상태 관리
- 이름 변경시 누적 기록 연쇄 업데이트
- 출전자 명단에서 일괄 회원 등록
- 월/연간 누적 벌금 추적

### 📋 대진표 (경기 전)
- 팀 배치 + 출전자 명단 입력
- AI 대진표 인식 (인쇄본 이미지 → 자동 추출)
- **🎲 랜덤 대진 생성**: 참가자·종목·인당 게임수 기반
  - 모든 가능한 경우의 수 나열 (균등 + 비균등)
  - 각 안마다 인당 배정 명시 (예: "남자 남복2+혼복2")
  - 사용자가 선택 후 생성
- 개인별 게임 배정 요약 실시간 표시

### 📝 결과 (경기 후)
- A4 출력본의 손글씨 점수/찬조금 AI 인식
- 25점 기준, 손글씨 숫자 특화 프롬프트
- 점수 입력시 승자/패자 자동 색상 표시

### 📄 A4 PDF 출력
- 원본 이미지와 동일한 12컬럼 표
- 빈 대진표 (경기 전) / 결과 포함 (경기 후) 2가지 모드
- 브라우저 인쇄 → PDF 저장 지원

### 🏆 순위 / 개인기록 / 누적 통계
- 승수 → 득실차 → 득점 순 정렬
- 종목별 개인 기록 추적
- 전체/올해/이번달 누적 통계

### 🤖 AI 비전 지원
- **Google Gemini** (무료, 매일 1,500회)
- **Anthropic Claude** (유료, 최고 정확도)
- **OpenAI GPT-4o** (유료)
- API 키는 브라우저 localStorage에만 저장 (외부 전송 없음)

## 사용법

1. `index.html` 파일을 브라우저에서 열기 (더블클릭)
2. **대회설정** 탭에서 일자/대회명 입력, AI 키 등록 (선택)
3. **회원** 탭에서 참가자 등록
4. **대진표** 탭에서:
   - 🎲 랜덤 생성 또는
   - 📷 AI 이미지 인식 또는
   - 직접 입력
5. 공유 탭에서 📄 A4 PDF 출력
6. 대회 진행 후 **결과** 탭에서 점수 입력 (수동 또는 AI 수기 인식)
7. **저장** 으로 해당 주차 누적

## 기술 스택

- Vanilla JavaScript (프레임워크 없음)
- LocalStorage (데이터 영속화)
- Tesseract.js (로컬 OCR, 폴백)
- html2canvas (이미지 내보내기)
- Gemini / Claude / OpenAI API (AI 비전)

## 데이터 보관

- 모든 데이터는 **브라우저 localStorage**에 저장
- JSON 백업/복원 기능 제공
- 서버 없음 → 프라이버시 완벽 보장
- 기기 간 이동시 JSON export/import 사용
